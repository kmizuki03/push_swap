## 📥 PDFに変換する3つの方法

### 方法1: ブラウザでPDF化（最も簡単・推奨）

1. HTMLファイルを開く
   ```bash
   # ファイルパス
   /home/user/push_swap/push_swap_algorithm_guide_ja.html
   ```

2. ブラウザで開く
   - Mac: Finder でダブルクリック
   - Windows: エクスプローラーでダブルクリック
   - Linux: `xdg-open push_swap_algorithm_guide_ja.html`

3. 印刷メニューを開く
   - Mac: ⌘ + P
   - Windows/Linux: Ctrl + P

4. 「PDFとして保存」を選択
   - 送信先: PDFに保存
   - ファイル名: push_swap_guide.pdf

✅ これで高品質なPDFが作成されます！

---

### 方法2: オンラインツール（インターネット接続が必要）

以下のサイトでMarkdownまたはHTMLファイルをアップロード：

**Markdownから:**
- https://www.markdowntopdf.com/
- https://md2pdf.netlify.app/
- https://dillinger.io/

**HTMLから:**
- https://www.sejda.com/html-to-pdf
- https://pdfcrowd.com/html-to-pdf/

使い方:
1. サイトにアクセス
2. ファイルをアップロード
3. 「Convert」または「変換」をクリック
4. PDFをダウンロード

---

### 方法3: コマンドライン（pandocがある場合）

```bash
# pandocのインストール
# Ubuntu/Debian:
sudo apt install pandoc texlive-xetex texlive-lang-japanese

# Mac:
brew install pandoc
brew install --cask basictex

# PDF生成
pandoc push_swap_algorithm_guide_ja.md \
  -o push_swap_guide.pdf \
  --pdf-engine=xelatex \
  -V CJKmainfont="Noto Sans CJK JP" \
  -V geometry:margin=1in

# または wkhtmltopdf を使用
sudo apt install wkhtmltopdf  # Linux
brew install wkhtmltopdf       # Mac

wkhtmltopdf push_swap_algorithm_guide_ja.html push_swap_guide.pdf
```

---

### 方法4: VSCodeで変換（VSCodeユーザー向け）

1. VSCodeで push_swap_algorithm_guide_ja.md を開く

2. 拡張機能をインストール:
   - "Markdown PDF" by yzane

3. コマンドパレット (Ctrl/⌘ + Shift + P) で:
   - "Markdown PDF: Export (pdf)" を選択

4. 自動的にPDFが生成されます
